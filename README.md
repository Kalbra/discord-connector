# Discord Connector
Adobe CC extension to add discord rich press to discord by using adobe CEP.


### Supported app
- [x] Photoshop 
- [x] Illustrator
- [x] Premiere Pro
- [x] InDesign
- [x] InCopy
- [x] Prelude
- [x] After Effects
- [x] Animate with issues
- [x] Audition with issus
- [x] Dreamweaver with isuses
