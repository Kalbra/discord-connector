# Discord Connector
Adobe CC extension to add discord rich press to discord by using adobe CEP


### Supported app
- [x] Photoshop 
- [x] Illustrator
- [ ] Premiere Pro
- [ ] InDesign
- [ ] InCopy
- [ ] Prelude
- [ ] After Effects
- [ ] Animate
- [ ] Audition
- [ ] Dreamweaver
